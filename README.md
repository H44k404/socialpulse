# SocialPulse

Futuristic social media management platform built with:
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React
- Recharts
- React Hook Form
- Zustand
- date-fns

## Getting started

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
